# Job Card for Vehicle Service

A web application to manage vehicle service job cards, track service details, and maintain vehicle service history.

## Features
- Create and manage job cards
- Vehicle and customer details
- Service status tracking
- Real-time notifications (optional)
- Bill generation and management
- Inventory tracking
- Frontend and backend separation

## Tech Stack
- Frontend: React
- Backend: Node.js / Express (serverless on Netlify)
- Database: MongoDB
